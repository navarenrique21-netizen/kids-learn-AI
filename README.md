# Kids Learning Spelling Game

A simple, mobile-first spelling game for kids built with vanilla HTML, CSS, and JavaScript.

## Features added
- ✅ Clean GitHub structure (index.html, style.css, script.js)
- ✅ Star success sound on correct answer (Web Audio, no files needed)
- ✅ Session timer (mm:ss)
- ✅ High score saved in localStorage
- ✅ Bilingual toggle EN ↔ ES (UI + word lists)
- ✅ Better shuffle (Fisher-Yates), no immediate word repeats
- ✅ Speech synthesis in correct language (en-US / es-MX)

## How to use
1. Clone or download this folder to your repo
2. Push to GitHub
3. Enable GitHub Pages (Settings → Pages → Deploy from main branch / root)
4. Open the Pages URL on phone or tablet

## Customize
- Edit `levelsEN` and `levelsES` in script.js to add your own words
- Change colors in style.css :root variables
